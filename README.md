# Convert pcd to pointcloud topic

## Usage



## ROS command

Use the command to convert pcd to ros topic:

```
rosrun pcl_ros pcd_to_pointcloud pcd_file_name.pcd 0.1 _frame_id:=/map cloud_pcd:=orig_cloud_pcd
```
